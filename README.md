Версия для JS5 в файле indexJS5.js

Сначала ход делает игрок. После этого ход делают все враги. Если герой рядом, то враг атакует, в противном случае он делает шаг на одну клетку в случайном направлении. Атака врага и героя происходят по направлениям вверх, вниз, влево, вправо (т.е без диагонали).

От себя добавил завершение игры при смерти героя или убийстве абсолютно всех врагов.

Иногда может возникнуть проблема, что герой или враг появился в закрытой комнате, в которую нет прохода. В таком случае обновить страницу.
